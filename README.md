# BFV-AimBot-V3

#Important

1. Download Lastest Version of Python
2. Intall it
3. Open CMD as admin 
4. write in CMD `pip install -r requirements.txt`
5. Done

#How To Start Hack
1. Edit `aim.bat` file and put your folder path after cd and Save it
2. Open `aim.bat` as Admin
3. Enjoy
